A high-performance, premium digital portfolio and service platform built with Next.js, React 19, and Tailwind CSS v4. Kalivox specializes in delivering scalable web, mobile, AI, and DevOps solutions for modern businesses.

## ✨ Key Features

- **🚀 Premium Aesthetics**: Modern, dark-themed UI with glassmorphism, dynamic gradients, and smooth scroll animations.
- **📱 Responsive Design**: Fully optimized for all screen sizes, from mobile to ultra-wide monitors.
- **⚡ Performance First**: Zero-layout shift, optimized image loading, and server-side rendering for lightning-fast speeds.
- **📧 Integrated Contact System**: Robust contact form powered by Nodemailer with SMTP integration.
- **🛠 Tech Stack Showcase**: Professional visualization of core competencies including AI/ML, Cloud, and Frontend/Backend technologies.
- **🔄 Scalable Architecture**: Built using a modular component-based architecture for easy extension.

## 🛠 Tech Stack

- **Frontend**: Next.js 15+ (App Router), React 19, TypeScript
- **Styling**: Tailwind CSS v4, Vanilla CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React, React Icons
- **Backend / Mail**: Next.js Server Actions, Nodemailer

## 🚀 Getting Started

### Prerequisites

- Node.js 18.x or higher
- npm, yarn, pnpm, or bun

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/umarfarooq57/portfolio.git
   cd portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Configure Environment Variables:**
   Create a `.env.local` file in the root directory and add your SMTP credentials:
   ```env
   SMTP_USER=your-email@gmail.com
   SMTP_PASS=your-app-specific-password
   ```

4. **Run the development server:**
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 📦 Deployment

The easiest way to deploy this project is via [Vercel](https://vercel.com/new).

1. Connect your GitHub repository to Vercel.
2. Ensure environment variables are set in the Vercel dashboard.
3. Deploy!
